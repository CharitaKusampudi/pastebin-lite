# Pastebin Lite

A minimal Pastebin-style app built with Next.js 16, Prisma, and PostgreSQL.

## Features
- Create text pastes
- View pastes via shareable URLs
- Optional expiration and view limits
- REST API endpoints

## Tech Stack
- Next.js (App Router)
- Prisma
- PostgreSQL
- TypeScript

## Getting Started
1. Clone the repo
2. Install dependencies
3. Set up `.env`
4. Run `npm run dev`
